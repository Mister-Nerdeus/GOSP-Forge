# Sponsorship, Finance, and Free Education

## Mission

Public school teachers and students should be able to use GOSP Forge’s core educational platform for free.

Commercial users, manufacturers, sponsors, grants, foundations, pro services, and in-kind support should fund the commons.

## Core statement

> Schools learn for free. Creators get credit. Sponsors support the commons. Manufacturers improve data quality. Nobody buys control.

## In-kind sponsorship

In-kind support should be tracked like money.

Ledger fields:

```text
sponsor
service/product
estimated value
expiration date
usage restrictions
lock-in risk
renewal risk
assigned owner
replacement plan
```

In-kind support is useful, but it should not create permanent lock-in.

## Sponsor independence

Sponsors may:

- fund development
- sponsor public challenges
- provide cloud/AI/hardware
- submit product specs
- participate in advisory discussions
- receive public recognition

Sponsors may not:

- control scoring
- own public modules
- suppress competitors
- control governance
- require exclusive integrations
- buy rankings
- hide sponsored status
- change open licenses retroactively

## Entity structure discussion

### Option A — Pure nonprofit

Pros:

- strong public trust
- grant-friendly
- school-friendly
- sponsor-friendly

Cons:

- admin burden
- slower setup
- less commercial flexibility

### Option B — For-profit

Pros:

- faster execution
- easier investment
- commercial services easier

Cons:

- weaker trust for open commons
- harder school/foundation funding

### Option C — Public Benefit Company

Pros:

- mission-aligned commercial structure
- flexible revenue

Cons:

- not a nonprofit
- less direct grant/donation fit

### Option D — Hybrid

Recommended long-term.

```text
GOSP Foundation
  open standards, education, governance, public module commons

GOSP Forge Services
  hosted pro tools, enterprise workspaces, advanced reports, support
```

## Recommended path

```text
1. Founder-led open-source project
2. Sponsor independence charter
3. Fiscal sponsorship exploration
4. Nonprofit/foundation if traction grows
5. Optional services company for sustainability
```

## Free public education policy

Public school teachers and students should receive:

- free access to core modules
- free educational projects
- free basic simulation
- free public product catalog
- free teacher/student challenge access
- sponsored AI support when available
- no surprise billing
- no student payment prompts

## Paid/pro features

Paid features may include:

- private commercial workspaces
- advanced reports
- high-volume batch simulation
- heavy compute
- professional estimate packages
- proprietary/internal module libraries
- priority support
- enterprise deployment
