# Module Registry

Registry entries include trust level, moderation status, license, attribution, and review history.

## Trust Levels

- `draft`: author-supplied or incomplete.
- `simulated`: has foundation simulation evidence.
- `fabricated`: has fabrication evidence.
- `tested`: has test evidence.
- `reviewed`: has human/domain review evidence.
- `field-used`: has declared field-use evidence, not a universal safety claim.
- `restricted`: visible but restricted; requires a safety reason.
- `deprecated`: retained for compatibility or audit.
- `removed`: removed from active use but retained as an audit record.

## Moderation Status

- `pending`: awaiting review.
- `approved`: visible under current policy.
- `restricted`: visible with restrictions; requires moderation reason.
- `removed`: removed from active use; requires moderation reason and audit record.

## Non-Claims

Registry trust does not certify potable water, provide professional engineering approval, approve production manufacturing, or prove real-world safety. Restricted and removed entries must preserve reasons so the audit trail is visible.
