# commons-sim Import Policy

Allowed after review: contracts, deterministic patterns, CLI ideas, graph primitives. Conditional: simulation formulas with assumptions exposed. Forbidden: old product framing as GOSP thesis, hidden constants, unlicensed content.

## Current Implementation Limits

This document describes foundation policy and contract intent. The current repository provides placeholder packages, example fixtures, and validation gates only. It does not claim production readiness or professional approval.
