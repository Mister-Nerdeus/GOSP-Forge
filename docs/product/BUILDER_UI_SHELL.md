# Builder UI Shell

The web package is a minimal read-only foundation shell. It loads the Automated Water Filter project manifest summary and shows project mode, ref counts, required refs, and safety boundary text.

## Boundaries

- No CAD editor is implemented.
- No persistence is implemented.
- No project editing is implemented.
- No potable-water certification, professional approval, or production manufacturing approval is provided.

## Validation

The package participates in workspace build and typecheck through Vite and TypeScript. It is intentionally limited to static inspection of bundled example data.
