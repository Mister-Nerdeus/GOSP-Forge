# Runtime Policy

Node 24 LTS is preferred and used by Docker/CI. This local bootstrap also allows Node 22 LTS because the current workstation reports Node v22.16.0. Runtime policy is centralized in package.json, .nvmrc, .tool-versions, and verify-runtime-version.mjs.

## Current Implementation Limits

This document describes foundation policy and contract intent. The current repository provides placeholder packages, example fixtures, and validation gates only. It does not claim production readiness or professional approval.
